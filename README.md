# LunaRover
Code that goes on the rover for the NASA 2020 Lunabotics Competition

To setup and run do: 
```shell
git clone https://github.com/UKyKORA/LunaRover.git
cd LunaRover
catkin_make
source devel/setup.bash
roslaunch drive drive_test.launch
```
